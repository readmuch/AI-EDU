# 직장인을 위한 AI 활용 교육 웹페이지

이 프로젝트는 **AI를 처음 접하는 직장인**이 기초 개념부터 실무 적용, 최종 개인 프로젝트 완성까지 단계적으로 학습할 수 있도록 만든 React 기반 단일 페이지 웹앱입니다.

실제 교육 서비스 랜딩페이지 형태를 목표로 구성했으며, 교육 대상/로드맵/커리큘럼/실습 예제/프롬프트 템플릿/FAQ까지 한 번에 확인할 수 있습니다.

## 주요 특징

- 3단계 교육 구조: **입문 → 실무 활용 → 프로젝트 완성**
- 직장인 업무 맥락 중심 콘텐츠(이메일, 회의록, 보고서, 데이터 분석, 기획 등)
- 실습 카드형 UI 및 단계별 난이도/예상 시간 제공
- 프롬프트 템플릿 복사 기능 제공
- CTA 버튼 클릭 시 해당 섹션으로 부드럽게 스크롤 이동
- 모바일/태블릿/데스크톱 반응형 레이아웃 지원

## 기술 스택

- React
- Vite
- Tailwind CSS

## 실행 방법

아래 명령어를 프로젝트 루트(`c:\dev\AI_Edu`)에서 순서대로 실행하세요.

### 1) 패키지 설치

```bash
npm install
```

### 2) 개발 서버 실행

```bash
npm run dev
```

실행 후 터미널에 표시되는 로컬 주소(예: `http://localhost:5173`)로 접속하면 페이지를 확인할 수 있습니다.

### 3) 프로덕션 빌드

```bash
npm run build
```

### 4) 빌드 결과 미리보기

```bash
npm run preview
```

## 제공 콘텐츠 요약

- Hero: 교육 제목/부제/CTA
- 교육 대상: 추천 대상 안내
- 교육 로드맵: 3단계 학습 흐름 카드
- 커리큘럼 상세: 단계별 목표, 이론, 실습, 학습 결과
- 실습 예제: 업무 상황별 연습 과제
- 프롬프트 템플릿: 즉시 사용 가능한 기본 양식
- 최종 프로젝트: 진행 단계 + 프로젝트 예시
- 기대 효과: 수강 전/후 비교 및 기대 성과
- FAQ: 자주 묻는 질문

## 참고 사항

- 본 프로젝트는 랜딩페이지 형태의 프론트엔드 앱입니다.
- 백엔드/DB 연동 없이 정적 콘텐츠 중심으로 구성되어 있습니다.
- 교육 신청 폼, 상담 신청, 사용자 추적(분석) 기능은 필요 시 확장 가능합니다.
